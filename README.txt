Приложение: "Книга контактов".

Приложение содержит:
	-список контактов
	-форма редактирования/удаления контакта
	-форма создания контакта

Контакт в книге содержит следующую основную информацию:
	-Фамилия
	-Имя
	-Отчество
	-Дата рождения
	-Номер телефона
	-Адрес
	-Электронный адрес

Проект реализован с применением технологий:
	-HTML5
	-CSS
	-JavaScript
	-AngularJS 1.6
	-Bootstrap 3

Запуск: Запустить файл app/view/sample.html в браузере
